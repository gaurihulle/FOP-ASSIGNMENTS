### FOP_LAB ASSIGNMENTS

## 📌 Overview
This repository contains **22 fundamental C programming assignments** designed to build a strong foundation in programming concepts.  
These assignments cover essential topics and help improve logical thinking and problem-solving skills.

---

## 🧠 Topics Covered
- Basic Input/Output
- Data Types & Variables
- Operators
- Conditional Statements (if-else, switch)
- Loops (for, while, do-while)
- Functions
- Arrays
- Strings
- Basic Problem Solving


---

## 🎯 Objectives
- Understand core concepts of C programming  
- Develop logical thinking  
- Practice coding through structured problems  
- Build a strong base for advanced programming

## 🛠️ Tools & Technologies
- C Programming Language  
- GCC Compiler / Turbo C / CodeBlocks
